## 关于博客

这个博客主要用于记录博主的个人成长的所有文章，在记录自己学习和思考的同时，希望来访的同学也能从中有所收获。

博客取名为「代码天涯路」，意即IT一行学习路漫漫，有很长的学习路径要走，但每一步的努力都不会白费，以此自勉。
